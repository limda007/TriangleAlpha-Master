# 快速入门

7 步完成从零到自动挂机的全流程。

## 第 1 步：启动主控

双击 `TriangleAlpha-Master.exe` 启动群控中心。首次启动会在用户目录下创建配置文件。

## 第 2 步：部署被控端

在每台挂机虚拟机上：

1. 将 `TriangleAlpha-Slave.exe` 放入 TestDemo 脚本同目录
2. 确保目录下存在 `SlaveClientConsole.exe`（可以是任意同名文件）
3. 在同目录创建 `主控IP.txt`，写入主控机器的 IP 地址
4. 运行 `TriangleAlpha-Slave.exe`

被控端启动后会自动连接主控，在大屏模式的节点表格中出现。

## 第 3 步：导入账号

在 **大屏模式** 点击「上传账号」，或在 **账号管理** 点击「导入账号」。

账号格式（每行一个）：
```
账号----密码----邮箱----邮箱密码----备注
```

其中邮箱、邮箱密码、备注为可选字段，最少只需 `账号----密码`。

## 第 4 步：配置验证码

在 **大屏模式 → 验证码 Tab**：

1. 点击「前往充值」跳转充值页面，完成充值
2. 将 API Key 粘贴到输入框
3. 点击「保存并下发」— Key 会自动推送到所有在线节点

## 第 5 步：下发配置

在 **大屏模式 → 配置 Tab**：

1. 设置补齐队友、武器、下号等级、舔包次数
2. 点击「下发配置」推送到所有在线节点

> **提示**：选中特定节点后，配置面板会自动回填该节点的当前配置，下发也仅针对选中节点。

## 第 6 步：启动挂机

在 **大屏模式 → 操作 Tab**，点击「启动/重启脚本」。

节点会自动请求账号，主控自动从账号池分配空闲账号并下发。等级达标后自动换号。

## 第 7 步：提取成果

账号完成后（状态变为「已完成」），在 **文件 Tab** 点击「提取账号」导出到文件。

如果启用了分销平台对接（设置页），已完成账号会自动上传到平台，无需手动提取。

---

**常见操作快捷流程**：

| 场景 | 操作 |
|------|------|
| 节点卡住 | 右键 → 停止脚本 → 启动脚本 |
| 批量重启 | 操作 Tab → 停止脚本游戏 → 启动/重启脚本 |
| 更换武器 | 配置 Tab → 选择武器 → 下发配置 |
| 检查余额 | 验证码 Tab → 查询按钮 |
| 节点分组 | 右键 → 设置分组 → 使用筛选下拉框查看 |
