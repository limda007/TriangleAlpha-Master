# 功能说明

## 🖥️ 大屏模式

主控中心的核心页面，一屏总览所有节点和账号状态。

### 📊 标题栏徽章

页面顶部有三个实时更新的徽章：

| 徽章 | 含义 | 举例 |
|------|------|------|
| 在线 | 在线节点数 / 总节点数 | 在线: 15/20 |
| 账号 | 可用账号数 / 总账号数 | 账号: 30/50 |
| 时长 | 主控运行了多久 | 2h35m |

### 📋 节点实时表格

显示所有被控端节点的实时状态。每一行代表一台挂机机器。

**状态圆点颜色说明：**

| 颜色 | 含义 |
|------|------|
| 🟢 绿色 | 节点在线，正常工作 |
| 🔴 红色 | 节点离线，无法连接 |
| ⚫ 灰色 | 节点断连，等待重连 |
| 🟡 黄色 | 其他状态 |

**表格列说明：**

| 列名 | 含义 |
|------|------|
| 机器名 | 被控端电脑的名称 |
| IP地址 | 被控端的网络地址 |
| 挂机账号 | 当前正在运行的游戏账号 |
| 等级 / 金币 | 当前账号的等级和金币数 |
| 运行时间 | 脚本已运行多长时间 |
| 运行状态 | 当前脚本状态（如有异常会显示⚠️警告） |
| CPU% / 内存% | 被控端的电脑资源使用情况 |

> 💡 **提示**：点击表头可以排序。支持 Ctrl+点击 多选节点。

### 🖱️ 右键菜单

在节点表格中，**右键点击**某一行可以看到操作菜单：

| 菜单项 | 作用 |
|--------|------|
| 复制 IP / 复制机器名 | 复制到剪贴板 |
| 启动/重启脚本 | 让这台机器开始跑脚本 |
| 停止脚本游戏 | 停止这台机器上的脚本和游戏 |
| 重启电脑 | 强制重启这台机器（要二次确认） |
| 分配账号 | 从账号池分配一个空闲账号 |
| 释放绑定账号 | 释放当前绑定的账号 |
| 设置分组 | 给节点设置分组名（方便筛选） |

### 📦 账号池

底部左侧区域，表格形式展示所有账号的实时状态。

**账号状态说明：**

| 状态 | 含义 |
|------|------|
| 空闲中 | 还没被分配给任何节点 |
| 运行中 | 已分配给节点，正在跑 |
| 已完成 | 达到目标等级，已完成 |
| 已取号 | 已被提取或被平台取走 |

底部统计栏显示：**总数、可用、运行中、已完成** 四个数字。

> 💡 **超时监控**：勾选后，如果某个节点长时间没有更新状态，主控会自动帮它重启脚本。适合处理脚本卡死的情况。

### 🎮 操作面板

底部右侧区域，分为 4 个页签：

#### 操作页签

| 按钮 | 作用 |
|------|------|
| 一键分发文件 | 选择文件，批量传输到所有节点 |
| 启动/重启脚本 | 让选中的（或全部在线）节点启动脚本 |
| 停止脚本游戏 | 停止选中的（或全部在线）节点的脚本 |
| 强制重启电脑 | 重启选中的（或全部在线）节点的电脑 |

> ⚠️ **注意**：如果没有选中任何节点，操作会对**全部在线节点**生效，系统会弹出确认窗口。

#### 文件页签

| 按钮 | 作用 |
|------|------|
| 提取账号 | 把已完成的账号导出到文件 |
| 导出所有 | 导出全部账号（不改变状态） |
| 批量删除文件 | 输入文件名，批量删除节点上的文件 |
| 清理单机账号 | 一键删除节点上的账号缓存文件 |
| 分发专属 Key | 选择 key.txt 文件分发到各节点 |

#### 配置页签

| 配置项 | 说明 |
|--------|------|
| 补齐队友 | 是否自动补满队伍（开启/关闭） |
| 武器配置 | 选择使用哪把武器 |
| 下号等级 | 达到多少级自动换号（默认 18 级） |
| 舔包次数 | 每局最多舔多少次包（默认 8 次） |

> 💡 **提示**：选中某个节点后，配置面板会自动显示该节点当前的配置值。点击「下发配置」会推送到选中节点。

#### 验证码页签

| 功能 | 说明 |
|------|------|
| 充值链接 | 点击可以打开充值网页 |
| API Key | 输入验证码识别服务的密钥 |
| 账户余额 | 显示剩余金额（绿色=充足，红色=不足） |
| 保存 | 只保存 Key 到本地电脑 |
| 保存并下发 | 保存 Key 并推送到所有在线节点 |

> 💡 **提示**：主控启动后，如果已保存了 API Key，会自动下发到缺失 Key 的节点，不需要手动操作。

## 👥 账号管理

独立的账号管理页面，可以导入、导出、查看所有账号。

### 统计卡片

页面顶部有 4 个大数字卡片：**总数、可用、运行中、已完成**，一眼就能看到账号池的整体情况。

### 工具栏按钮

| 按钮 | 作用 |
|------|------|
| 导入账号 | 从 .txt 文件导入账号 |
| 提取账号 | 导出已完成账号并标记为已取号 |
| 导出所有 | 导出全部账号到文件 |
| 清空 | 删除全部账号（需要确认） |

右侧下拉框可以按状态筛选：**全部、空闲中、运行中、已完成、已取号**。

### 表格操作

- **点击密码列**：可以切换显示/隐藏密码
- **右键菜单**：可以复制账号、密码、邮箱等信息
- **多选释放**：选中多行后，右键可以批量释放绑定

> 💡 **账号格式**：`账号----密码----邮箱----邮箱密码----备注`（四个短横线分隔，邮箱和备注可省略）

## 📋 操作历史

记录所有对节点执行过的操作，方便回看和排查问题。

| 列名 | 含义 |
|------|------|
| 时间 | 什么时候执行的 |
| 操作类型 | 做了什么（启动脚本/停止脚本/重启电脑等） |
| 目标节点 | 对哪些节点操作的 |
| 详情 | 具体参数 |
| 结果 | 执行结果 |

- 上方下拉框可以**按操作类型筛选**
- 点击「清空历史」可以**清除所有记录**

## 📄 实时日志

远程查看被控端的运行日志，支持同时看多个节点。

### 工具栏

| 控件 | 作用 |
|------|------|
| 搜索框 | 输入关键词过滤日志 |
| 级别筛选 | 选择只看 INFO / WARN / ERROR |
| 自动滚动 | 打开后新日志自动滚到最底下 |
| 清空 | 清除所有日志 |

### 左侧节点列表

显示所有上报过日志的节点名称。**勾选哪个就看哪个的日志**。点击「全选」可以同时看所有节点。

### 右侧日志区

日志格式：`[节点名] [时间] [级别] 内容`

> 💡 **提示**：每个节点最多保留 5000 行日志，不会占用太多内存。

## ⚙️ 设置

修改后**自动保存**，无需手动确认。

### 网络设置

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| UDP 监听端口 | 8888 | 接收被控端心跳和状态的端口 |
| TCP 指令端口 | 9999 | 向被控端发送命令的端口 |
| TCP 日志端口 | 8890 | 接收被控端日志的端口 |
| 心跳间隔 | 3 秒 | 被控端多久上报一次心跳 |
| 离线超时 | 15 秒 | 超过这个时间没收到心跳就标记离线 |

> ⚠️ **注意**：修改端口后需要**重启主控**才能生效。

### 外观设置

| 设置项 | 说明 |
|--------|------|
| 主题 | 浅色模式 / 深色模式 / 跟随系统 |
| 云母效果 | 窗口半透明效果（仅 Windows 11） |

### 分销平台

| 设置项 | 说明 |
|--------|------|
| 启用平台对接 | 打开后自动上传已完成账号 |
| API 地址 | 平台的网址 |
| 用户名 / 密码 | 平台的登录账号 |
| 分组名称 | 账号上传到平台的哪个分组 |

> 💡 **提示**：启用后，已完成的账号会自动上传。平台取号后，本地状态也会自动更新为「已取号」。
