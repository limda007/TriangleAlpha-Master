# 功能说明

## 大屏模式

主控中心的核心页面，一屏总览所有节点和账号状态。

### 标题栏徽章

页面顶部显示三个实时徽章：

| 徽章 | 含义 |
|------|------|
| 在线 | 在线节点数 / 总节点数 |
| 账号 | 可用账号数 / 总账号数 |
| 时长 | 主控运行时长（自启动起计） |

### 节点实时表格

显示所有被控端节点的实时状态，共 15 列：

| 列名 | 说明 |
|------|------|
| (状态圆点) | 绿色=在线，红色=离线，灰色=断连，黄色=其他 |
| 机器名 | 被控端上报的计算机名 |
| IP地址 | 被控端 IP |
| 挂机账号 | 当前运行的游戏账号 |
| 等级 | 当前账号等级 |
| 金币 | 当前账号金币 |
| 运行时间 | 脚本运行时长 |
| 运行状态 | 脚本当前状态（带 Token 告警标记） |
| CPU% | 被控端 CPU 使用率 |
| 内存% | 被控端内存使用率 |
| 版本 | 被控端 Slave 版本号 |
| 补齐队友 | 当前补队配置（开启/关闭） |
| 武器配置 | 当前武器模板 |
| 下号等级 | 达标换号的等级阈值 |
| 舔包次数 | 每局舔包上限 |

- **分组筛选**：表格上方的下拉框可按分组过滤节点
- **排序**：点击表头可按该列排序
- **多选**：支持 Ctrl/Shift 多选行

### 右键菜单

在节点表格中右键单击可执行以下操作：

- **复制 IP** / **复制机器名** — 复制到剪贴板
- **启动/重启脚本** — 向该节点发送启动指令
- **停止脚本游戏** — 停止该节点的脚本和游戏
- **重启电脑** — 强制重启该节点（需二次确认）
- **分配账号** — 从账号池分配一个空闲账号给该节点
- **释放绑定账号** — 释放该节点当前绑定的账号
- **设置分组** — 为该节点设置分组名称

### 账号池

底部左侧区域，展示所有账号的实时状态表格：

| 列名 | 说明 |
|------|------|
| 账号 | 用户名 |
| 密码 | 遮蔽显示 |
| 邮箱 | 绑定邮箱 |
| 邮箱密码 | 遮蔽显示 |
| 状态 | 空闲中/运行中/已完成/已取号 |
| 分配机器 | 绑定的节点名 |
| 等级 | 当前等级 |
| 金币 | 当前金币 |
| 上传时间 | 导入时间 |
| 完成时间 | 完成时间 |

底部统计栏显示：总数、可用、运行中、已完成。

**超时监控**：勾选后，停滞超过指定分钟数的节点将自动重启脚本。

### 操作面板

底部右侧区域，分为 4 个 Tab 页：

#### 操作 Tab

| 按钮 | 功能 |
|------|------|
| 一键分发文件 | 选择文件批量传输到节点脚本目录 |
| 启动/重启脚本 | 向选中/全部在线节点发送启动指令 |
| 停止脚本游戏 | 停止选中/全部在线节点的脚本和游戏 |
| 强制重启电脑 | 强制重启选中/全部在线节点（需二次确认） |

#### 文件 Tab

| 按钮 | 功能 |
|------|------|
| 提取账号 | 导出已完成账号到文件并标记为已取号 |
| 导出所有 | 导出全部账号（不改变状态） |
| 批量删除文件 | 输入文件名列表，批量删除节点上的文件 |
| 清理单机账号 | 删除节点上的 accounts.json 等文件 |
| 分发专属 Key | 选择 key.txt 文件分发到节点 |

#### 配置 Tab

| 配置项 | 说明 |
|--------|------|
| 补齐队友 | 开启/关闭自动补满队伍 |
| 武器配置 | 选择武器模板推送到节点 |
| 下号等级 | 达标后自动换号的等级阈值 |
| 舔包次数 | 每局舔包上限 |

选中节点时自动回填该节点当前配置，点击「下发配置」推送到选中/全部在线节点。

#### 验证码 Tab

| 功能 | 说明 |
|------|------|
| 充值链接 | 点击跳转到充值页面 |
| API Key | 输入验证码识别服务的 Key |
| 账户余额 | 实时查询余额（品字型布局：总额 + 付费/积分） |
| 保存 | 仅保存 Key 到本地 |
| 保存并下发 | 保存 Key 并推送到选中/全部在线节点的 token.txt |

> **提示**：启动时如主控已保存 API Key，会自动下发到缺失或 Key 不一致的节点。

## 账号管理

独立的账号管理页面，提供完整的账号生命周期管理。

### 统计卡片

页面顶部 4 个统计卡片：**总数**、**可用**、**运行中**、**已完成**。

### 工具栏

| 按钮 | 功能 |
|------|------|
| 导入账号 | 从文件导入（格式：账号----密码----邮箱----邮箱密码----备注） |
| 提取账号 | 导出已完成账号并标记为已取号 |
| 导出所有 | 导出全部账号 |
| 清空 | 清空所有账号（需确认） |

右侧下拉框可按状态筛选：全部、空闲中、运行中、已完成、已取号。

### 表格

10 列表格，与大屏模式账号池相同。点击密码/邮箱密码列可切换显示/遮蔽。

### 右键菜单

- **复制账号** / **复制密码** / **复制邮箱** / **复制邮箱密码** / **复制整行**
- **释放绑定** — 释放选中的运行中账号（支持多选）

---

## 操作历史

记录所有对节点执行的操作，方便回溯和审计。

### 表格

| 列名 | 说明 |
|------|------|
| 时间 | 操作执行时间 |
| 操作类型 | 启动脚本/停止脚本/重启电脑/下发配置 等 |
| 目标节点 | 操作对象（节点名或数量） |
| 详情 | 操作的具体参数 |
| 结果 | 执行结果 |

- **类型筛选**：下拉框过滤特定操作类型
- **清空历史**：清除所有记录

---

## 实时日志

远程查看被控端的 Console 输出，支持多节点聚合查看。

### 工具栏

| 控件 | 功能 |
|------|------|
| 搜索框 | 按关键词过滤日志内容 |
| 级别筛选 | 全部级别 / INFO / WARN / ERROR |
| 自动滚动 | 开启后新日志自动滚动到底部 |
| 清空 | 清除所有日志缓存 |

### 节点列表

左侧列表显示所有上报过日志的节点，支持多选和全选。仅显示选中节点的日志。

### 日志区

右侧显示格式化的日志输出：`[节点名] [时间] [级别] 内容`

每个节点最多保留 5000 行日志。

---

## 设置

应用配置页面，修改后自动保存。

### 网络设置

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| UDP 监听端口 | 8888 | 接收被控端心跳和状态 |
| TCP 指令端口 | 9999 | 向被控端发送指令 |
| TCP 日志端口 | 8890 | 接收被控端日志 |
| 心跳间隔 | 3 秒 | 被控端发送心跳的间隔 |
| 离线超时 | 15 秒 | 超时无心跳标记为离线 |

### 外观设置

| 设置项 | 说明 |
|--------|------|
| 主题 | 浅色 / 深色 / 跟随系统 |
| 云母效果 | 开启/关闭窗口云母效果 |

### 分销平台

| 设置项 | 说明 |
|--------|------|
| 启用平台对接 | 开启后自动上传已完成账号 |
| API 地址 | 平台 API 根地址 |
| 用户名 | 平台登录用户名 |
| 密码 | 平台登录密码 |
| 分组名称 | 上传账号所属分组 |

启用后，已完成的账号会自动上传到分销平台，平台取号后本地状态自动更新。
